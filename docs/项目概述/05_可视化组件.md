# 可视化组件

CZSC 框架提供两套可视化方案：一套是基于 Streamlit 的 Web 展示组件库（`czsc/svc/`），另一套是基于 Plotly/ECharts 的编程式绘图工具（`czsc/utils/plotting/`、`czsc/utils/echarts_plot.py`）。

## Streamlit 可视化组件库 — `czsc/svc/`

提供 70+ 个 Streamlit 展示函数，用于在 Streamlit Web 应用中快速构建策略分析界面。所有组件通过 `czsc.svc` 命名空间延迟加载。

### 收益可视化 — `returns.py`

展示策略收益相关的图表组件：

- `show_cumulative_returns`：累计收益曲线
- `show_daily_return`：日收益率分布
- `show_drawdowns`：回撤分析图
- 其他收益相关展示函数

### 相关性可视化 — `correlation.py`

展示因子和品种间相关性的图表组件：

- `show_correlation`：相关性矩阵热力图
- `show_sectional_ic`：截面 IC 分析
- `show_ts_rolling_corr`：滚动时间序列相关性

### 因子分析可视化 — `factor.py`

展示因子研究和分层分析结果的组件：

- `show_feature_returns`：特征收益分析
- `show_factor_layering`：因子分层收益
- `show_event_return`：事件驱动收益分析

### 回测可视化 — `backtest.py`

展示回测结果的综合性组件：

- `show_weight_backtest`：权重回测结果展示
- `show_holds_backtest`：持仓回测结果展示
- `show_comprehensive_weight_backtest`：综合权重回测展示

### 统计可视化 — `statistics.py`

展示统计分布和分析结果的组件：

- `show_yearly_stats`：年度统计汇总
- `show_psi`：PSI（群体稳定性指标）
- `show_normality_check`：正态性检验

### 策略可视化 — `strategy.py`

展示策略运行状态和优化结果的组件：

- `show_czsc_trader`：CzscTrader 策略状态展示
- `show_optuna_study`：Optuna 参数优化结果展示
- `show_portfolio`：投资组合展示

### 权重可视化 — `weights.py`

展示持仓权重相关的组件：

- `show_weight_ts`：权重时间序列
- `show_weight_dist`：权重分布
- `show_weight_cdf`：权重累积分布

### 价格敏感性分析 — `price_analysis.py`

展示价格变化对策略收益影响的敏感性分析组件。

### 表单组件 — `forms.py`

提供交互式表单组件，包括代码编辑器表单（`code_editor_form`）和权重回测配置表单（`weight_backtest_form`）。

### 工具函数 — `utils.py`

提供 Streamlit 应用运行工具（`streamlit_run`）。

## 编程式绘图工具

### Plotly 绘图 — `czsc/utils/plotting/`

提供基于 Plotly 的编程式绘图功能，适用于脚本和 Jupyter Notebook 环境。

#### K线图表 — `kline.py`

- **KlineChart**：K线图表类，支持绘制标准K线图、缠论标注（分型、笔、中枢）、信号标记等
- `plot_czsc_chart`：一键绘制缠论分析图表

#### 回测图表 — `backtest.py`

回测结果相关的 Plotly 图表，包括累计收益、回撤、月度热力图等。

#### 权重图表 — `weight.py`

权重相关的 Plotly 图表展示。

#### 通用绘图 — `common.py`

提供公共绘图函数，包括 `figure_to_html`（图表转 HTML）、`add_year_boundary_lines`（添加年度分隔线）等复用工具。

### ECharts 交互式图表 — `czsc/utils/echarts_plot.py`

基于 pyecharts 和 lightweight-charts 的交互式图表：

- `kline_pro`：专业级交互式K线图
- `trading_view_kline`：TradingView 风格的K线展示
