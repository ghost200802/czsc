# 工具库

工具库提供项目所需的通用功能支撑，包括数据处理、技术指标、缓存管理、可视化绘图、报告生成等。代码位于 `czsc/utils/` 目录，按功能划分为多个子模块。

## 模块结构

```
czsc/utils/
├── __init__.py              # 通用工具函数聚合
├── analysis/                # 分析工具（相关性、事件、统计）
├── crypto/                  # 加密工具
├── data/                    # 数据管理（缓存、客户端、转换、验证）
├── plotting/                # 可视化绘图
├── backtest_report.py       # 回测报告
├── bi_info.py               # 笔信息计算
├── cross.py                 # 截面排名
├── echarts_plot.py          # ECharts 可视化
├── features.py              # 特征工程工具
├── html_report_builder.py   # HTML 报告生成器
├── index_composition.py     # 指数成分查询
├── io.py                    # IO 工具
├── kline_quality.py         # K线质量检查
├── log.py                   # 日志工具
├── mark_czsc_status.py      # 缠论状态标记
├── oss.py                   # 阿里云 OSS
├── pdf_report_builder.py    # PDF 报告生成器
├── sig.py                   # 信号工具函数
├── ta.py                    # 技术指标封装
├── trade.py                 # 交易工具
├── warning_capture.py       # 警告捕获
├── weights_convert.py       # 权重转换
└── word_writer.py           # Word 文档生成
```

## 通用工具函数 — `czsc/utils/__init__.py`

提供全局通用的工具函数，包括数值舍入（`x_round`）、Python 命名空间获取（`get_py_namespace`）、动态导入（`import_by_name`）、K线周期排序（`freqs_sorted`）、网格参数生成（`create_grid_params`）等。

## 技术指标封装 — `czsc/utils/ta.py`

对 TA-Lib 和自研指标的 Python 封装，提供统一接口调用常用技术分析指标，包括 MACD、RSI、BOLL、KDJ、ATR、SMA、EMA、WMA 等。

## 数据管理 — `czsc/utils/data/`

### DataClient — `client.py`

统一的数据获取客户端，支持配置数据源 URL 和 Token，提供标准化的数据查询接口。

### DiskCache — `cache.py`

基于磁盘的缓存管理器，支持自动缓存数据查询结果，避免重复请求。提供 `disk_cache` 装饰器和 `home_path`（缓存目录）、`clear_cache`（清空缓存）等工具函数。

### 数据转换与验证 — `converters.py`、`validators.py`

提供数据格式转换和输入数据质量验证功能。

## 分析工具 — `czsc/utils/analysis/`

- **corr.py**：相关性分析工具，支持截面 IC、滚动相关性等计算
- **events.py**：事件分析工具，用于评估事件驱动策略的收益特征
- **stats.py**：统计工具，包括滚动日绩效（`rolling_daily_performance`）、盈亏平衡点计算（`cal_break_even_point`）等

## 交易工具 — `czsc/utils/trade.py`

提供交易相关的辅助函数，包括日收益率重采样（`resample_to_daily`）、无风险收益率获取（`risk_free_returns`）、K线数据更新（`update_bbars`）等。

## 截面排名 — `czsc/utils/cross.py`

提供截面排名工具 `cross_sectional_ranker`，用于多品种截面策略中按指标对品种进行排名。

## 可视化绘图 — `czsc/utils/plotting/`

详见 [05_可视化组件.md](05_可视化组件.md)。

## 报告生成

### 回测报告 — `backtest_report.py`

生成策略回测的文本/HTML 报告，汇总关键绩效指标。

### HTML 报告生成器 — `html_report_builder.py`

基于 Streamlit/Plotly 的交互式 HTML 回测报告生成器，支持自定义报告内容和样式。

### PDF 报告生成器 — `pdf_report_builder.py`

基于 reportlab 的 PDF 回测报告生成器，支持生成可打印的策略分析报告。

### Word 文档生成 — `word_writer.py`

基于 python-docx 的 Word 文档生成工具。

## K线质量检查 — `kline_quality.py`

检查K线数据质量，包括缺失值检测、异常值识别、数据连续性验证等。

## 缠论状态标记 — `mark_czsc_status.py`

在K线数据上标记缠论分析状态（如当前笔方向、中枢位置等），便于可视化展示。

## 笔信息计算 — `bi_info.py`

计算笔的详细信息，包括笔的长度、斜率、力度等统计指标。

## 指数成分查询 — `index_composition.py`

查询股票指数的成分股信息。

## 信号工具 — `sig.py`

信号相关的辅助函数，用于信号的计算、转换和验证。

## 特征工程工具 — `features.py`

提供特征工程的辅助函数，用于因子研究和特征构建。

## ECharts 可视化 — `echarts_plot.py`

基于 pyecharts 和 lightweight-charts 的交互式K线图表，提供 TradingView 风格的K线展示（`kline_pro`、`trading_view_kline`）。

## IO 工具 — `io.py`

数据序列化工具，基于 dill 实现（`dill_dump`、`dill_load`），以及 JSON 文件读写（`read_json`、`save_json`）。

## 阿里云 OSS — `oss.py`

封装阿里云对象存储服务（OSS）的文件上传下载功能（`AliyunOSS` 类）。

## 日志工具 — `log.py`

基于 loguru 的日志配置和管理。

## 权重转换 — `weights_convert.py`

持仓权重格式的转换工具。

## 加密工具 — `czsc/utils/crypto/fernet.py`

基于 Fernet 对称加密的数据加密/解密工具，用于敏感信息保护。

## 警告捕获 — `warning_capture.py`

Python 警告信息的捕获和管理工具。
