# 飞书服务集成

飞书服务模块提供与飞书开放平台的深度集成能力，支持消息推送、电子表格操作、多维表格管理等。代码位于 `czsc/fsa/` 目录。

## 模块概述

飞书服务模块（FSA — Feishu Service API）封装了飞书开放平台的核心 API，使 CZSC 策略能够通过飞书进行实时监控通知和数据管理。通过 `czsc.fsa` 命名空间延迟加载。

## 核心功能

### 即时消息（IM）— `im.py`

通过飞书机器人发送消息通知。

主要功能：
- 文本消息推送（`push_text`）
- 卡片消息推送（`push_card`）：支持飞书卡片 JSON 2.0 结构，包含丰富的交互组件
- 支持富文本、图表、表格等多种卡片内容

卡片消息支持三大类组件：
- **展示类**：标题、文本、富文本、图片、图表、表格、分割线等
- **交互类**：输入框、按钮、下拉选择、日期选择器等
- **容器类**：分栏、表单容器、折叠面板等

详见 `docs/飞书卡片2.0文档.md`。

### 电子表格 — `spreed_sheets.py`

对飞书电子表格的读写操作。

主要类：
- **SpreadSheets**：电子表格管理，支持创建、查询电子表格
- **SingleSheet**：单个工作表操作，支持数据读写、格式设置

### 多维表格 — `bi_table.py`

对飞书多维表格（Bitable）的读写操作。

主要类：
- **BiTable**：多维表格管理，支持记录的增删改查

### 策略信息推送 — `push_strategy_latest.py`

将策略的最新运行状态、持仓信息和绩效指标推送到飞书。

主要功能：
- 自动格式化策略数据为飞书卡片
- 定时或事件触发的策略状态推送

### API 基础类 — `base.py`

飞书 API 的基础封装类（`FeishuApiBase`），提供统一的请求认证、错误处理和重试机制，供其他模块继承使用。
