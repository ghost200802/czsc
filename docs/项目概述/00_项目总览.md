# CZSC 项目总览

## 项目简介

CZSC（缠中说禅）是一个基于缠论技术分析理论的量化交易研究框架，采用 Python + Rust 混合架构，提供从数据获取、缠论分析、信号生成、策略回测到绩效评估的完整量化交易研究链路。

- **版本**：0.10.12（2026-03-08），正在向 1.0.X 正式版演进
- **Python 要求**：>= 3.10
- **许可证**：Apache-2.0
- **仓库地址**：https://github.com/waditu/czsc

## 技术架构

项目采用 Python + Rust 混合架构：

- **Rust 层**（`rs_czsc`）：负责性能关键的缠论分析器、K线合成器、权重回测、核心数据对象等
- **Python 层**：负责上层业务逻辑、信号系统、策略框架、可视化组件、外部集成等
- **延迟加载**：顶层模块通过 `__getattr__` 实现按需导入，避免不必要的初始化开销

## 模块划分

项目核心代码位于 `czsc/` 目录下，按功能划分为以下模块：

| 模块 | 目录 | 职责 |
|------|------|------|
| 核心分析 | `czsc/core.py`、`czsc/py/` | 缠论数据结构、分析引擎、K线合成 |
| 信号系统 | `czsc/signals/` | 200+ 预定义信号函数 |
| 交易与回测 | `czsc/traders/` | 策略执行、回测引擎、绩效分析 |
| 工具库 | `czsc/utils/` | 通用工具、数据处理、绘图、报告 |
| 可视化组件 | `czsc/svc/` | Streamlit Web 可视化组件 |
| 数据连接器 | `czsc/connectors/` | 第三方数据源与交易框架接入 |
| 飞书服务 | `czsc/fsa/` | 飞书消息推送、电子表格、多维表格 |
| 特征因子 | `czsc/features/` | 时序特征因子（收益率、量价因子） |
| 研究工具 | `czsc/sensors/`、`czsc/eda.py` | CTA 研究、探索性数据分析 |
| 策略基类 | `czsc/strategies.py` | 策略开发模板与基类 |

## 核心工作流

```
数据获取(DataClient) → K线标准化(format_standard_kline) → 缠论分析(CZSC)
    → 信号生成(CzscSignals) → 事件组合(Event) → 交易决策(CzscTrader/Position)
    → 回测(WeightBacktest/DummyBacktest) → 绩效评估(daily_performance)
    → 可视化(svc/) → 报告(HTML/PDF/Word)
```

## 缠论对象层级

```
RawBar（原始K线）→ NewBar（去包含处理）→ FX（分型）→ BI（笔）→ ZS（中枢）
```

## 信号-事件-交易流水线

- **Signal（信号）**：基础技术指标或市场状态的判定结果
- **Event（事件）**：多个信号的逻辑组合（AND/OR/NOT），定义交易触发条件
- **Position（仓位）**：完整的交易策略单元，包含开仓和平仓规则
