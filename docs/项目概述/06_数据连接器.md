# 数据连接器

数据连接器模块提供与第三方数据源和交易框架的对接能力，支持从不同渠道获取行情数据或接入实盘交易。代码位于 `czsc/connectors/` 目录。

## 模块概述

连接器模块采用统一接口设计，每个连接器负责一个外部数据源或交易框架的对接。通过 `czsc.connectors` 命名空间导入，支持延迟加载。

## 连接器列表

### Tushare 数据连接器 — `ts_connector.py`

对接 Tushare 金融数据接口，获取A股市场行情数据。

主要功能：
- 获取股票日K线和分钟K线数据
- 获取交易日历和股票列表
- 支持复权处理（前复权/后复权/不复权）
- 需配置 Tushare Token

### 天勤量化连接器 — `tq_connector.py`

对接天勤量化（TqSdk）框架，获取期货市场行情数据。

主要功能：
- 获取期货品种的Tick和K线数据
- 支持实盘交易指令下发
- 支持多品种数据订阅

### CCXT 加密货币连接器 — `ccxt_connector.py`

对接 CCXT（CryptoCurrency eXchange Trading Library）框架，获取加密货币交易所行情数据。

主要功能：
- 支持数百个加密货币交易所的统一接口
- 获取数字货币的K线、Ticker、深度数据
- 支持实盘交易接口

### Research 数据连接器 — `research.py`

内置的研究数据连接器，提供用于策略研究和回测的样本数据。

主要功能：
- 提供本地或远程的研究数据集
- 支持数据缓存和增量更新
- 用于开发和测试阶段的数据模拟
